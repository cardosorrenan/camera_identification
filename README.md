- Instalar pipenv
```
  pip3 install pipenv
```
- Entrar no ambiente
```
  pipenv shell 
```
- Executar script
```
  python3 main.py
```
- Instalar/Desinstalar algum pacote
```
  pipenv install/uninstall *
```
- Extrair requirements.txt
```
  pipenv run pip3 freeze > requirements.txt
```
- Instalar apartir de requirements.txt
```
  pipenv install -r requirements.txt
```
- Sair do ambiente pipenv
```
  exit
```
